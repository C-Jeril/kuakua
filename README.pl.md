## Główne Moduły

### 🎮 Gry (`/games`)

Różnorodna kolekcja interaktywnych gier do rozrywki, relaksacji i stymulacji poznawczej.

| Kategoria | Przykłady |
|-----------|-----------|
| Puzzle | 2048, Sudoku, Puzzle, Kostka Rubika |
| Poznawcze | Pamięć, N-Back, Stroop, Czas reakcji |
| Relaksacja | Malowanie, Muzyka, Guidowana medytacja |
| Klasyki Google | Snake, Pacman, Kółko-krzyżyk, Pasjans |

### 📋 Testy Psychologiczne (`/test`)

Naukowo ugruntowane testy psychologiczne do samooceny.

| Typ | Opis |
|-----|------|
| Testy walidowane | Ugruntowane instrumenty z silnym wsparciem empirycznym (PHQ-9, GAD-7, Big Five) |
| Testy adaptowane | Zmodyfikowane lub skrócone wersje inspirowane walidowanymi instrumentami |
| Testy eksploracyjne | Narzędzia autorefleksji inspirowane mechanizmami psychologicznymi |

**Uwaga**: Wszystkie testy zawierają odpowiednie zastrzeżenia i są przeznaczone do celów edukacyjnych i autorefleksji, nie do diagnozy klinicznej.

### 🧪 Eksperymenty Poznawcze (`/experiment`)

Interaktywne implementacje klasycznych paradygmatów poznawczych i zadań eksperymentalnych.

| Paradygmat | Opis |
|------------|------|
| Stroop | Test interferencji kolor-słowo |
| N-Back | Zadanie pamięci roboczej |
| IGT | Iowa Gambling Task |
| Flanker | Test uwagi selektywnej |
| Go/No-Go | Test kontroli inhibicyjnej |

### 🔧 Narzędzia (`/tools`)

## Psychologia Pozytywna

Kuakua opiera się na zasadach **psychologii pozytywnej**, dziedziny naukowej poświęconej badaniu tego, co czyni życie wartym przeżycia.

### Kluczowe Koncepcje

| Koncepcja | Opis |
|-----------|------|
| **Eudaimonia** | Ludzki rozkwit, realizacja potencjału |
| **Subiektywny dobrostan** | Satysfakcja z życia, pozytywne emocje |
| **Mocne charakteru** | Pozytywne cechy takie jak wdzięczność, nadzieja, odporność |
| **Flow** | Stan optymalnego zaangażowania w działania |
| **Odporność** | Zdolność adaptacji w obliczu przeciwności |

### Korzyści

- 🌟 **Poprawa dobrostanu** - Skupienie na optymalnym funkcjonowaniu
- 💪 **Rozwój mocnych stron** - Kultywacja pozytywnych cech
- 🤝 **Pozytywne relacje** - Budowanie znaczących połączeń
- 🎯 **Cel i znaczenie** - Poszukiwanie wartościowych celów
- 🧘 **Uważność** - Uważność i świadomość

Kuakua ma na celu budowanie **psychologicznie ugruntowanych, etycznie odpowiedzialnych i globalnie dostępnych narzędzi i treści**, które pomagają ludziom lepiej się zrozumieć i poprawić swój dobrostan.

Wierzymy w:
- ✅ Ważność psychologiczną i zapobieganie szkodzie
