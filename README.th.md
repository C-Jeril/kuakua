## โมดูลหลัก

### 🎮 เกม (`/games`)

คอลเลกชันเกมอินเทอร์แอคทีฟที่หลากหลายเพื่อความบันเทิง, การผ่อนคลาย และการกระตุ้นทางปัญญา

| หมวดหมู่ | ตัวอย่าง |
|----------|----------|
| เกมปริศนา | 2048, Sudoku, จิ๊กซอว์, ลูกบาศก์รูบิค |
| ทางปัญญา | หน่วยความจำ, N-Back, Stroop, เวลาปฏิกิริยา |
| การผ่อนคลาย | วาดภาพ, ดนตรี, สมาธินำทาง |
| คลาสสิก Google | Snake, Pacman, โอเอกซ์, Solitaire |

### 📋 แบบทดสอบทางจิตวิทยา (`/test`)

แบบทดสอบประเมินตนเองทางจิตวิทยาที่มีหลักฐานทางวิทยาศาสตร์

| ประเภท | คำอธิบาย |
|--------|----------|
| แบบทดสอบที่ผ่านการตรวจสอบ | เครื่องมือที่ยอมรับแล้วพร้อมหลักฐานเชิงประจักษ์ที่แข็งแกร่ง (PHQ-9, GAD-7, Big Five) |
| แบบทดสอบที่ปรับปรุง | เวอร์ชันที่ดัดแปลงหรือสั้นลงโดยอิงจากเครื่องมือที่ผ่านการตรวจสอบ |
| แบบทดสอบเชิงสำรวจ | เครื่องมือสะท้อนตนเองโดยอิงจากกลไกทางจิตวิทยา |

**หมายเหตุ**: แบบทดสอบทั้งหมดมีข้อความปฏิเสธความรับผิดชอบที่เหมาะสม และมีไว้เพื่อวัตถุประสงค์ทางการศึกษาและการสะท้อนตนเอง ไม่ใช่เพื่อการวินิจฉัยทางคลินิก

### 🧪 การทดลองทางปัญญา (`/experiment`)

การนำเสนอแบบอินเทอร์แอคทีฟของกระบวนทัศน์ทางปัญญาแบบดั้งเดิมและงานทดลอง

| กระบวนทัศน์ | คำอธิบาย |
|-------------|----------|
| Stroop | แบบทดสอบการรบกวนสี-คำ |
| N-Back | งานหน่วยความจำทำงาน |
| IGT | Iowa Gambling Task |
| Flanker | แบบทดสอบความใส่ใจเลือก |
| Go/No-Go | แบบทดสอบการควบคุมการยับยั้ง |

### 🔧 เครื่องมือ (`/tools`)

เครื่องมือเพื่อสุขภาวะและประสิทธิภาพในชีวิตประจำวัน

- **Focus Flow** - เทคนิค Pomodoro พร้อมการติดตาม
- **กระดานคะแนน** - บันทึกคะแนน
- **นับคำ** - เครื่องมือเขียน
- **ตัวจับเวลา** - การจัดการเวลา

### 📚 เนื้อหา (`/docs`, `/blog`, `/explore`)

ฐานความรู้ที่ครอบคลุมเกี่ยวกับจิตวิทยาและสุขภาวะ

| ส่วน | เนื้อหา |
|------|----------|
| เอกสาร | ทฤษฎีทางจิตวิทยา, แนวคิด, การวิจัย |
| บล็อก | บทความเกี่ยวกับสุขภาวะ, การพัฒนาตนเอง |
| สำรวจ | แหล่งข้อมูลที่คัดสรร, เว็บไซต์, แอปพลิเคชัน |

## จิตวิทยาเชิงบวก

Kuakua ตั้งอยู่บนหลักการของ **จิตวิทยาเชิงบวก** ซึ่งเป็นสาขาวิทยาศาสตร์ที่มุ่งศึกษาสิ่งที่ทำให้ชีวิตมีคุณค่า

### แนวคิดหลัก

| แนวคิด | คำอธิบาย |
|--------|----------|
| **Eudaimonia** | การเบ่งบานของมนุษย์, การบรรลุศักยภาพ |
| **สุขภาวะอัตวิสัย** | ความพึงพอใจในชีวิต, อารมณ์เชิงบวก |
| **จุดแข็งของบุคลิกภาพ** | คุณลักษณะเชิงบวก เช่น ความกตัญญู, ความหวัง, ความยืดหยุ่น |
| **Flow** | สถานะของการมีส่วนร่วมอย่างเหมาะสมในกิจกรรม |
| **ความยืดหยุ่น** | ความสามารถในการปรับตัวเมื่อเผชิญอุปสรรค |

### ประโยชน์

- 🌟 **ปรับปรุงสุขภาวะ** - มุ่งเน้นการทำงานอย่างเหมาะสมที่สุด
- 💪 **พัฒนาจุดแข็ง** - บ่มเพาะคุณลักษณะเชิงบวก
- 🤝 **ความสัมพันธ์เชิงบวก** - สร้างการเชื่อมโยงที่มีความหมาย
- 🎯 **จุดมุ่งหมายและความหมาย** - แสวงหาเป้าหมายที่มีคุณค่า
- 🧘 **สติ** - การตระหนักรู้และความใส่ใจ

## พันธกิจของเรา

Kuakua มีเป้าหมายที่จะสร้าง **เครื่องมือและเนื้อหาที่มีหลักฐานทางจิตวิทยา, มีความรับผิดชอบทางจริยธรรม และเข้าถึงได้ทั่วโลก** ที่ช่วยให้ผู้คนเข้าใจตนเองได้ดีขึ้นและปรับปรุงสุขภาวะของพวกเขา

เราเชื่อใน:
- ✅ ความถูกต้องทางจิตวิทยาและการป้องกันอันตราย
- ✅ ความชัดเจนทางแนวคิดและโครงสร้าง
- ✅ ความเข้าใจและการเข้าถึงของผู้ใช้
- ✅ ความแข็งแกร่งทางเทคนิคและการบำรุงรักษา
- ✅ การใช้งานหลายภาษาและความไวต่อวัฒนธรรม
