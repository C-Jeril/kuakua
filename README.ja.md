## ポジティブ心理学の歴史

ポジティブ心理学は、精神的健康と幸福を向上させることを目的とする新興分野です。その目的は、個人が自分の強みを発見し、それを活用することで、より高いレベルの生活満足度と喜びを達成することです。

ポジティブ心理学の発展は 20 世紀末に始まり、1997 年に正式に導入されました。この分野は、2000 年 1 月に Martin Seligman と Mihaly Csikszentmihalyi によって発表された「Positive Psychology: An Introduction」によって認知されました。

心理学者の Martin Seligman と彼の同僚によって推進されたポジティブ心理学は、心理学の分野における革命的な変化を表し、人類社会の歴史における新しいマイルストーンを示しています。それは東西の文化の知恵と心理学研究を結びつけます。

Seligman と彼の同僚は、心理学は精神疾患や問題の研究を超えて、精神的健康と幸福を向上させる方法を探求すべきだと主張しています。ポジティブ心理学は、幸福を研究するための科学的方法を使用し、この分野においてポジティブな指向を推進し、ポジティブな心理特性、幸福、調和の取れた発展を育成することに重点を置いています。

ポジティブ心理学の中心概念は「エウダイモニア（Eudaimonia）」であり、これはアリストテレス哲学の用語で、最高の人間の善を意味し、しばしば「繁栄」または「良い生活」と翻訳されます。ポジティブ心理学者は、主観的幸福（SWB）と幸福という用語をしばしば同義語として使用します。

この分野は、個人と社会の幸福を向上させることを優先し、「生活の質を向上させることを目指して、ポジティブな主観的経験、ポジティブな個人特性、およびポジティブな制度」を研究します。

彼らは、配偶者、家族、友人、同僚、広範なネットワークとの社会的関係、クラブや社会組織への所属、運動および瞑想の実践など、さまざまな要因が幸福と主観的幸福に寄与すると信じています。

精神性もまた、幸福を向上させる要因と見なされています。研究者は、精神的な実践や宗教的な献身を、幸福の潜在的な源泉およびポジティブ心理学の不可欠な部分として研究し続けています。

財政的収入はある程度まで幸福を増加させることができますが、特定の閾値を超えると、その効果は平坦になるか、さらには減少する可能性があります。ポジティブ心理学は、個人のポジティブな資質、成長の可能性、そして自分の運命を形作る自由を育成することを強調しています。

## ポジティブ心理学は私に何をしてくれますか？

ウェルビーイングと最適な機能の向上 ポジティブ心理学は、症状を軽減するだけでなく、個人のウェルビーイングと最適な機能を向上させることに焦点を当てており、従来の心理学を補完し、代替するものではありません。ポジティブ心理学の一般的なテーマには、人生を楽しむこと、感謝、親切、ポジティブな関係の促進、希望と意味の追求が含まれます。

ウェルビーイングに焦点を当てる 研究によると、ウェルビーイングと精神病理学は中程度に相関しているが、精神的健康の独立した構成要素です。精神病理学の治療が成功した後でも、低いレベルのウェルビーイングが持続し、心理的苦痛の重大なリスク要因となることがあります。

心理的症状の緩和 高いレベルのウェルビーイングは、症状の再発や再発を含む心理的症状を緩和し、生活の質と寿命を向上させることができます。したがって、臨床サンプルは、ポジティブな感情、認知、または行動を明示的に向上させることを目的としたポジティブ心理学的介入（PPI）から大いに利益を得ることができます。

補助的な役割 ポジティブ心理学は、幸福、抑うつ、または不安に関連するストレスを緩和する上で有意な効果を持たないが、ウェルビーイングを向上させることによって心理的苦痛を間接的に緩和する能力は広く認識されています。

ポジティブ心理学の主な利点 人生を楽しむこと：ポジティブな経験を楽しみ、感謝すること。感謝：人生の良いことを認識し、感謝する

こと。親切：他者に対して思いやりと利他主義を実践すること。ポジティブな関係の促進：健康的で支援的な関係を築き、維持すること。希望と意味の追求：目標を追求し、人生の目的を見つけること。これらの側面に焦点を当てることで、ポジティブ心理学は個人がより充実し、回復力のある生活を送るのを助けます。
